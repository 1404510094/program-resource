# PHP编程风格指南

参考：[PSR-1: Basic Coding Standard](https://www.php-fig.org/psr/psr-1/)、[PSR-2: Coding Style Guide](https://www.php-fig.org/psr/psr-2/)

* 类名`StudlyCaps`
* 函数名`camelCase`
* 关键字`true`, `false`, `null`必须小写
* 字符编码 **必须** `UTF-8`无`BOM`
* 缩进采用`4个空格`，不要使用`Tab`
* 每行最多`80个字符`
* 在下面情况必须空一行：
   * `namespace`语句结束
   * `use`语句块结束
* 函数定义，花括号必须另起一行
* 使用`elseif`，不要使用`else if`
